# Stage 1 - Fundamentals (1.5-2 months)

## Weeks 1-2

- Types, stack vs heap, addresses, references and pointers.
- Exercise folder: `exercises/week01_memory`.

## Weeks 3-4

- RAII, constructors, destructors, copy semantics.
- Exercise folder: `exercises/week02_raii`.

## Weeks 5-8

- `unique_ptr`, `shared_ptr`, `weak_ptr`, ownership design.
- Exercise folder: `exercises/week03_smart_pointers`.

## Rule

Do not write non-trivial code with raw `new` and `delete`.
