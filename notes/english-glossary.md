# C++ and RE English Glossary (Starter 50)

## Core C++ (1-20)

1. **variable** [вэриэбл] — переменная; именованное место для значения.
2. **type** [тайп] — тип; определяет, какие данные хранятся и как.
3. **value** [вэлью] — значение; содержимое переменной.
4. **scope** [скоуп] — область видимости; где имя существует в коде.
5. **lifetime** [лайфтайм] — время жизни; когда объект создан и уничтожен.
6. **stack** [стэк] — стек; автоматическая память функции.
7. **heap** [хип] — куча; динамическая память.
8. **address** [эдрэс] — адрес; место объекта в памяти.
9. **pointer** [поинтер] — указатель; переменная с адресом.
10. **reference** [референс] — ссылка; второе имя объекта.
11. **array** [эрэй] — массив; набор элементов одного типа.
12. **function** [фанкшн] — функция; блок кода, который можно вызвать.
13. **parameter** [пэрэметер] — параметр; входное значение функции.
14. **argument** [аргьюмент] — аргумент; фактическое значение при вызове.
15. **return value** [ритёрн вэлью] — возвращаемое значение функции.
16. **class** [класс] — класс; шаблон для объектов.
17. **object** [обджект] — объект; экземпляр класса.
18. **member** [мембер] — член класса; поле или метод.
19. **method** [мэсэд/метод] — метод; функция внутри класса.
20. **constructor** [констрактор] — конструктор; вызывается при создании объекта.

## Resource and Ownership (21-35)

21. **destructor** [дестрактор] — деструктор; вызывается при уничтожении объекта.
22. **RAII** [ар-эй-ай-ай] — ресурс привязан к жизни объекта.
23. **ownership** [оунершип] — владение; кто отвечает за освобождение ресурса.
24. **resource** [рисорс] — ресурс; память, файл, сокет, mutex, handle.
25. **copy constructor** [копи констрактор] — конструктор копирования.
26. **copy assignment** [копи ассайнмент] — оператор копирующего присваивания.
27. **move constructor** [мув констрактор] — конструктор перемещения.
28. **move assignment** [мув ассайнмент] — оператор перемещающего присваивания.
29. **deep copy** [дип копи] — глубокая копия; копируются сами данные.
30. **shallow copy** [шэллоу копи] — поверхностная копия; копируется адрес.
31. **smart pointer** [смарт поинтер] — умный указатель.
32. **unique_ptr** [юник поинтер] — уникальное владение объектом.
33. **shared_ptr** [шэрд поинтер] — разделяемое владение объектом.
34. **weak_ptr** [уик поинтер] — слабая ссылка без владения.
35. **memory leak** [мемори лик] — утечка памяти; память выделена и не освобождена.

## Build, Debug, and RE (36-50)

36. **compiler** [компайлер] — компилятор; переводит код в машинный.
37. **linker** [линкер] — линковщик; связывает объектные файлы и библиотеки.
38. **build** [билд] — сборка; процесс компиляции и линковки.
39. **Debug build** [дебаг билд] — сборка для отладки.
40. **Release build** [релиз билд] — оптимизированная сборка для релиза.
41. **warning** [ворнинг] — предупреждение компилятора.
42. **error** [эррор] — ошибка компиляции/линковки.
43. **breakpoint** [брейкпоинт] — точка останова в отладчике.
44. **call stack** [кол стэк] — стек вызовов функций.
45. **register** [реджистер] — регистр CPU.
46. **instruction** [инстракшн] — инструкция процессора.
47. **disassembly** [дисассембли] — дизассемблирование (асм-представление бинаря).
48. **process** [процесс] — процесс; запущенная программа.
49. **thread** [тред] — поток выполнения внутри процесса.
50. **handle** [хэндл] — дескриптор Windows-объекта.
