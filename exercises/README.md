# Exercises

- `week01_memory` - stack, heap, addresses.
- `week02_raii` - Buffer class, ctor/dtor, copy assignment.
- `week03_smart_pointers` - ownership with `unique_ptr` and `shared_ptr`.
- `week04_stl` - word frequency with `unordered_map`.
- `week05_algorithms` - `sort`, `find_if`, `transform`.
- `week06_move_semantics` - move-only type.
- `week07_winapi` - process list and open process.
- `week08_pe_asm` - basic binary and assembly exploration.
- `week09_re_lab` - pointer chains in a safe local lab.
- `week10_dll_hooking` - controlled DLL and hook demo.
