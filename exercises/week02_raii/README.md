# Week 02 - RAII, Constructors, Destructors, Copy/Move

## Goal

Understand object lifetime and ownership by reading constructor/destructor logs.

## Build target

`week02_raii_lifecycle`

## What to watch in output

- Constructor when object is created.
- Copy constructor when object is copied.
- Copy assignment when assigning between existing objects.
- Move constructor and move assignment with `std::move`.
- Destructor order at scope exit.
